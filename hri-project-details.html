MECH438 Human Robot Interface

Project Overview

Traditional gaming experiences often lack the real-world tactile interaction that can enhance user immersion. Integrating force feedback into gaming environments presents an opportunity to improve engagement, realism, and even educational or rehabilitative applications. Additionally, understanding the trade-off between speed and accuracy in gameplay can provide valuable insights into human-computer interaction dynamics.

Solution

Our team developed a maze game with force feedback, where players navigate a virtual maze while experiencing haptic resistance upon hitting obstacles. The system includes:

  • A haptic development device controlled via STM32, generating force feedback perpendicular to maze walls.
  • A custom physics model that calculates force magnitude and direction based on obstacle proximity.
  • A Python-based user interface, designed using OpenCV, displaying only the end-effector for a streamlined experience.
  • A data-driven analysis of speed vs. accuracy, using Spearmanâ€™s Rank Correlation to determine behavioral trends in gameplay.

Design Considerations

  • Realistic Force Simulation: The haptic feedback system was calibrated using stiffness and damping adjustments to mimic real-world interactions.
  • User Experience Optimization: A practice round was included before testing to ensure fair results.
  • Quantitative Evaluation: Gameplay data was analyzed statistically to validate the correlation between speed and accuracy.

Impact

This force feedback-enhanced maze game bridges the gap between virtual gaming and real-world interaction, offering a more immersive and engaging experience. Beyond entertainment, the technology has potential applications in rehabilitation and cognitive training, providing an interactive and adaptive platform for motor skill development.
