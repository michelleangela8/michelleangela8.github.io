10.022 Designing Energy Systems

Project Overview

Conventional Peltier dehumidifiers often run continuously, leading to unnecessary energy consumption and reduced efficiency. This project aims to enhance dehumidification performance by implementing automated humidity control and optimizing airflow, making the system more efficient in maintaining comfortable relative humidity levels.

Solution

Developed a sensor-integrated Peltier dehumidifier that automatically regulates power based on ambient humidity. Using a humidity sensor and bang-bang control, the system switches off when the target humidity is reached, reducing power consumption. Additionally, airflow optimization was tested by replacing restrictive plastic grills with a mesh to improve heat dissipation and condensation efficiency.

Design Considerations

  • Energy Optimization: Used an aluminum disk for optimal conductivity and a strontium-iron oxide magnet for strong, stable magnetic fields.
  • Airflow Efficiency: Replaced restrictive plastic grills with a 3D-printed mesh to enhance cooling performance.
  • Form Factor & Usability: Prototype included temporary wiring and breadboards, with future integration planned for improved portability.

Impact

This project demonstrates a cost-effective, energy-efficient approach to improving Peltier dehumidifiers, reducing unnecessary power usage while maintaining optimal indoor humidity levels. Future enhancements include integrating a vibration motor to improve condensation runoff and further miniaturizing components for better portability.
